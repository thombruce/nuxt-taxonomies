## Installation

Add TNT to your project:

```sh
yarn add https://github.com/thombruce/tnt --dev
```

Add TNT to buildModules in `nuxt.config.js`:

```js
buildModules: [
  '@thombruce/tnt'
]
```

## Overwrite Default Styles

```css
@import "@thombruce/tnt";

/* Declare your styles or local imports here. */
```
